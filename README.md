# 🎮 Agarz Personalizado

**Versión personalizada del juego Agarz.com con diseño épico rojo y negro.**

## 📋 Descripción

Juego multijugador tipo Agar.io con temática oscura y épica. Basado en el código original de richirm/agarz.

## ✨ Características

- ✅ Formulario de login con diseño épico (silueta + luna roja)
- ✅ Mundo del juego con colores rojo y negro
- ✅ Mecánicas de juego tipo Agar.io
- ✅ Diseño personalizado
- ✅ Tema oscuro y épico

## 🚀 Instalación

1. Clona este repositorio
   ```bash
   git clone https://github.com/abelith-hub/agarz-personalizado.git
   ```

2. Abre `index.html` en tu navegador

3. ¡A jugar! 🎮

## 🎨 Diseño

El juego cuenta con:
- Fondo rojo/oscuro épico
- Silueta de personaje en pantalla de login
- Interfaz con colores personalizados
- Efectos visuales oscuros
